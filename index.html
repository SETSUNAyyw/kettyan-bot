Commands コマンドリスト 指令集

  • /start - Greet with kettyan and start using. けっちゃんと挨拶をします。 打個招呼

  • /help - See help message. 使い方を見ます。 查看使用方法

  • /r0 - Randomly pick one illust id from Pixiv. Theoretically, the illust will be safe for work, but in case that the illust is R-18, a #NSFW (not safe for work) tag may be added in the caption. Pixiv のイラストがランダムに表示されます。もし18禁画像が出てきたら、#NSFW（職場では見ない方が良い）のタグが付けられますのでご注意を。 從Pixiv隨機獲取一張插畫。有一定機率會出現18禁（并會被標記為#NSFW）的圖像。

    The illust id may contain multiple images, and all of them (no more than 10) will be sent to the user. If the total number of images is larger than 10, only the first 10 of them will be sent. The returned images are in compressed jpg format, and original files can be accessed from the pixiv link attached with the images. Pixivのイラストには複数の画像が含まれている場合、10枚以下の画像しか送られません。画像はjpg形式で圧縮されて送られます。付けたリンクを開くとPixivへ飛んで、元ファイルをダウンロードできます。 Pixiv的一個插畫ID可能會包含多張圖像，而K醬一次最多能上傳10張。圖像會以jpg壓縮格式傳回，單擊附帶的連結也可直接從Pixiv下載原始圖像。

    Time to fetch the images depends on server conditions and varies from 1s to 20s. If no images are returned after 30s, the command will be dumped due to timeout. サーバーや回線の状態によって時間がかかりますので返信が来るまで少々お待ちを。通常は20秒以内に必ず届けますが、30秒以上かかったらリクエストが無効になり、タイムアウトになります。タイムアウトが発生した場合は、新しいコマンドをリクエストしてください。 根據伺服器以及網路連接狀態，K醬會在1至20秒之内進行回覆。如果超過30秒未收到回覆，説明連結已經超時，需要重新發送指令。

Get illust by pixiv id Pixiv IDでイラストをダウンロードする 用Pixiv ID獲取圖片

What is a pixiv id?

A pixiv illust id is the unique identifier for the illust. The pixiv id may be found in the pixiv url.

e.g. url: https://www.pixiv.net/artworks/12345678, id: 12345678

e.g. url: https://www.pixiv.net/member_illust.php?mode=medium&illust_id=33445566, id: 33445566

Pixiv IDとは？

PixivイラストのIDはその唯一の識別子です。イラストのURLにはIDが記載されています。

例： URL: https://www.pixiv.net/artworks/12345678, ID: 12345678

例： URL: https://www.pixiv.net/member_illust.php?mode=medium&illust_id=33445566, ID: 33445566

什麽是Pixiv ID？

每個Pixiv插畫都有唯一的ID，在URL裡就可以找到。

例： URL: https://www.pixiv.net/artworks/12345678, ID: 12345678

例： URL: https://www.pixiv.net/member_illust.php?mode=medium&illust_id=33445566, ID: 33445566

When a number is sent to kettyan, kettyan will regard it as a pixiv id and try to return the images of this illust id. Kettyan will tell the user if the pixiv id is invalid (no such id). 数字のみが送られた場合は、Pixiv IDとして認識されます。もちろん、すべての数字が有効なPixiv IDというわけではありません。https://www.pixiv.net/artworks/xxxxxxxxが存在している場合だけです。 向K醬DM數字信息，會被識別為Pixiv ID。但并不是所有數字都是合法的Pixiv ID，只有真實存在的插畫才會被下載并返回。

URLs with https://www.pixiv.net/artworks/xxxxxxxx format can also be sent to kettyan, which will be recognized as a pixiv id. Forwarded messages are also supported as long as the url is contained in the text and no images are sent together with the text (images are handled in here). 代わりに、https://www.pixiv.net/artworks/xxxxxxxxのURLをそのままけっちゃんに転送すると検出することもできます。画像がテキストと一緒に送られない限り、転送メッセージにはURL以外の内容があったとしても構いません。 此外，可以直接發送帶有https://www.pixiv.net/artworks/xxxxxxxx格式的URL，同樣能被識別。訊息中含有除圖像之外的文本都沒有問題（若DM了圖像，就會觸發反向圖像搜索）。

Search pixiv illust by tags or keywords タグやキーワードでイラストを検索する 搜尋標籤或關鍵字

If the message contains words, kettyan will recognize them as tags. Tags are separated by space (not comma or semicolon). Keyword search is also available, and only one keyword is supported, i.e., if multiple keywords separated by space are sent, then only the first one will be taken. 数字以外のテキストメッセージはタグまたはキーワードとして扱われています。タグはカンマ（,）またはセミコロン（;）でなく、スペース（␣）で区切られます。キーワードは一つしか取られませんので、スペースで区切られた複数のキーワードを送っても最初のキーワードだけが有効になります。 除數字之外的文本訊息都會被識別為標籤或者關鍵字。標籤用空格（␣）分隔，請不要用逗號（,）或是分號（;）代替。關鍵字則只能為一個單詞，若有多個關鍵字將取第一個。

Since tags/keyword search all work in text mode, when a text message is sent to kettyan, tags search will occur first, and then keyword search is triggered if tags search returns nothing. タグとキーワード検索はどちらもDMテキストによってトリガーされるため、タグモードが先に行われ、結果が見つかりません場合はキーワードモードに切り替わります。 由於標籤和關鍵字搜尋是一樣由文本訊息觸發的，文本將先被當作多個標籤處理，若找不到結果再會改成關鍵字搜尋。

Keyword search is not very accurate, so simple keywords such as "gawr" are recommended. Tags search is more powerful and recommended, though it is not guaranteed to find the required illust especially with too many tags. キーワード検索は少し弱いので、シンプルな単語（「妹ちゃん」とか）をお願いします。代わりに、タグ検索がより頻繁に使われていますし、お勧めします。もちろん、タグが多すぎて検索結果がほとんど出ない場合もあります。 關鍵字搜尋比較不準確，不是簡單詞匯（例如【妹ちゃん】）一般會搜尋不到結果，因此建議使用標籤搜尋。當然，標籤太多的話也很難找到相應的圖像。

Find similar illust 類似のイラストを探す 搜尋類似的圖片

When photo message is sent, kettyan will search it on Saucenao. If the message contains multiple images, kettyan will serach for each of them independently. A link with similarities will be returned and the preview of the link will be provided by Telegram if possible. 画像を直接に送ると、けっちゃんはSaucenaoでこの画像に似ているイラストを探していきます。複数の画像をまとめて送ってもOKです。返信メッセージは下記のように、類似度やリンクが返され、可能であればプレビューがTelegramによって提供されます。 向K醬DM圖像，就可以觸發Saucenao提供的反向圖像搜尋。發送一個包含多張圖像的媒體組也是OK的。如下圖所示，回覆會包含原始連結以及類似度，Telegram客戶端也會提供網址的圖像預覽。

Three interactive buttons will be attached to the returned link. リンクの下には三つのボタンが付けられています。 訊息下方有三個可互動的按鈕。

If you think the search result is satisfying, you may press the ⭕ button or just leave it without doing anything; If the result is unexpected, you may try to press ❌ to see next result with lower similarity. When no other results can be shown, a Yandex search link will be provided. If too many people are searching, then the yandex link will be unavailable due to bot detection. 探したいイラストが見つかった場合は、良ければ⭕を押すか、またはそのまま放置していても大丈夫です。❌を押すと、さらに他の検索結果が次々と出てきます。しかし、類似度はだんだん低くなります。全部ハズレたら、Yandexのイメージ検索リンクが送られます。リクエストが多くなり、ボットがYandexに検知されたり、リンクが表示できなくなる場合もあります。 如果你覺得已經找到了你想要的圖片，那麽單擊⭕即可，或者放著不管。如果你覺得不是很滿意，可以嘗試單擊❌查看下一條結果（但類似度會更低）。所有結果都顯示完畢后，若仍未搜尋到想要的圖像，你可以單擊最後的Yandex連結。由於請求過多會被Yandex的反機器人檢測到，因此這個連結不一定會出現。

If you are searching for an incomplete photo's original version, saucenao seems to be poor at such work, so it may be better to send a complete one. If kettyan returns a similar illust but not a pixiv one, you may try to press この画像で検索する to further search for this image. 画像の一部分だけで検索するのはSaucenaoにとって少し厳しそうですから、画像全体を送信した方が楽になります。この画像で検索するを押すと、検索結果でもう一度検索できます。 另外，Saucenao搜尋部分圖片并不準確，最好能夠提供全圖。單擊この画像で検索する，可以用當前的結果再次進行搜尋。

The buttons will expire in about 1h, and then nothing will happen when pressed. ボタンの有効期限がおよそ1hになりますので、切ったら押しても何も起こらないのです。 按鈕在大約1h之内過期，過期之後再單擊則不會有反應。

Convert Telegram stickers to images TelegramスタンプをPNG/JPGに変換する 轉換表情貼圖

Send telegram sticker to kettyan and the png original file together with the compressed jpg picture will be returned. スタンプを送ると、pngオリジナルファイルやjpg圧縮画像に変換できます。 向K醬發送貼圖，將會返回其png原始圖像以及jpg壓縮圖像。

Only support static stickers, i.e., animated stickers cannot be converted using kettyan. 動くスタンプはまだ変換できません。 動畫貼圖暫時不支持轉換。
